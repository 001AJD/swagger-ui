Plugin API

Plugin API

A plugin is an object that contains functions and components capable of modifying and augmenting Swagger-UI's functionality.

Format

A plugin may contain any or all of these keys:

const MyPlugin = {
  statePlugins: {
    anyStateKey: {
      actions,
      reducers,
      selectors,
      wrapActions,
      wrapSelectors
    }
  },
  components: {

  },
  wrapComponents: {

  }
}
